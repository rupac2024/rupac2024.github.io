Specificity

Specificity refers to which CSS rule is selected when multiple rules' selector refers to an element with specified selector. For example, a rule can be made for an element, id, and class that an element has defined. However, the same properties are changed in the CSS; therefore, one rule must be referred. Specificity solves this problem by creating a hierarchy.

This text will be red becasue inline CSS has the highest specificity.
Specificity order (high to low):
1. Inline
2. ID
3. Classes, pseudo-classes, attribute selectors
4. Elements and pseudo-elements
However, if you use the "!important" after a CSS rule, then that rule will be of highest priority (for the specific property). That is why the background color is purple.
